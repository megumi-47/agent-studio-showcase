02 · PLATFORM FEATURES

平台能力，逐项演示。

每段 15–40 秒；滚动到某段自动播放，可拖动进度、全屏查看。

01
FIG 01 · ROUTE

意图路由分支

换一条目的地类输入（济州岛），Switch 路由自动进入目的地子流程——各意图分支都是独立的 Python 函数。

02
FIG 02 · TRACES

调用链回放

每一步的输入输出、耗时、token 成本自动留存；模型调用有独立明细，badcase 可一键转为评测用例。

03
FIG 03 · EVAL

评测系统

用例集批量运行，自动评分加人工复核；改 prompt 前后，效果有量化对比。

04
FIG 04 · EXPERIMENT

AB 实验与灰度

变体、白名单、灰度比例都在平台内配置，调整即时生效，不需要重新发布。

05
FIG 05 · MODELS

模型中心

模型统一接入管理，按业务方统计用量与成本；对比 Playground 支持 1–3 个模型同题并排（DeepSeek / Qwen 跨厂商选型），比 TTFT、token 与耗时。

06
FIG 06 · IDE

IDE 与项目组织

工作流、prompt、技能按多文件项目组织，Web IDE 内直接浏览编辑；保存即热加载、多节点自动同步，也可绑定 git 仓库。

07
FIG 07 · DOCS

使用文档

配套使用手册与工作台同仓维护：平台总览、10 分钟上手、开发/运营工作区指南、API 参考，工作台内一键直达。
